# Special Methods in Python

1. `__getitem__()` — item access, e.g. `my_bag[0]`
2. `__setitem__()` — item assignment, e.g. `my_bag[0] = "laptop"`
3. `__contains__()` — membership testing, e.g. `"pen" in my_bag`
4. `__call__()` — callable objects, e.g. `my_object("hello")`
5. `__iter__()` / `__next__()` — iteration support, e.g. `for` loops
6. `__bool__()` — boolean evaluation, e.g. `bool(my_bag)`
7. `__del__()` — destructor, runs when an object is deleted
